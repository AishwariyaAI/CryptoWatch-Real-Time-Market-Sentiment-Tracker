## 📌 Project Overview

CryptoWatch analyzes cryptocurrency news headlines in real time and generates actionable market intelligence using Natural Language Processing (NLP) and Large Language Models (LLMs).

The platform aggregates crypto news, evaluates market sentiment, tracks Bitcoin and Ethereum prices, monitors the Fear & Greed Index, generates AI-powered summaries, and provides trading recommendations through an interactive dashboard.

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/AishwariyaAI/CryptoWatch-Real-Time-Market-Sentiment-Tracker.git
cd CryptoWatch-Real-Time-Market-Sentiment-Tracker
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Create a `.env` file:

```env
GROQ_API_KEY=your_groq_api_key
NEWS_API_KEY=your_newsapi_key
```

Run the application:

```bash
streamlit run app.py
```
